1. Thay đổi nội dung

Nhấn nút để thay đổi nội dung này.

Giải thích chi tiết:

  1. Chúng ta sử dụng hai biến toàn cục:
    • originalText: để lưu trữ nội dung ban đầu của phần tử.
    • isChanged: để theo dõi trạng thái hiện tại của nội dung (đã thay đổi hay chưa).
  2. Hàm changeText() thực hiện các bước sau:
    • Tìm phần tử có id "demo" bằng document.getElementById().
    • Kiểm tra xem phần tử có tồn tại không. Nếu không, hiển thị lỗi và kết thúc hàm.
    • Nếu nội dung chưa được thay đổi (!isChanged):
      • Lưu nội dung hiện tại vào originalText.
      • Thay đổi nội dung thành "Nội dung đã được thay đổi!".
      • Đặt isChanged thành true.
    • Nếu nội dung đã được thay đổi:
      • Khôi phục nội dung ban đầu từ originalText.
      • Đặt isChanged thành false.
    • Ghi log nội dung hiện tại ra console để dễ dàng theo dõi.

Giải thích

Ứng dụng: Thao tác DOM và quản lý trạng thái

Trong ví dụ này, chúng ta sử dụng JavaScript để thay đổi nội dung của một phần tử HTML khi người dùng nhn nút, và cho phép khôi phục nội dung ban đầu khi nhấn lại.


let originalText = ""
let isChanged = false

function changeText() {
    const demoElement = document.getElementById("demo")
    
    if (!demoElement) {
        console.error("Không tìm thấy phần tử có id 'demo'")
        return
    }
    
    if (!isChanged) {
        originalText = demoElement.innerHTML
        demoElement.innerHTML = "Nội dung đã được thay đổi!"
        isChanged = true
    } else {
        demoElement.innerHTML = originalText
        isChanged = false
    }
    
    console.log("Nội dung hiện tại:", demoElement.innerHTML)
}

Ví dụ này minh họa cách JavaScript xử lý input người dùng và thực hiện phép tính đơn giản.


Về hàm solveQuadraticEquation():

  • Đầu tiên, hàm kiểm tra xem input có hợp lệ không bằng cách sử dụng isNaN().
  • Nếu a = 0, hàm xử lý các trường hợp đặc biệt:
    • Nếu b = 0 và c = 0: Phương trình vô số nghiệm.
    • Nếu b = 0 và c ≠ 0: Phương trình vô nghiệm.
    • Nếu b ≠ 0: Giải phương trình bậc nhất bx + c = 0.
  • Nếu a ≠ 0, hàm tính delta và xử lý ba trường hợp:
    • Delta > 0: Hai nghiệm phân biệt.
    • Delta = 0: Nghiệm kép.
    • Delta < 0: Hai nghiệm phức.
  • Kết quả được làm tròn đến 2 chữ số thập phân bằng toFixed(2).

Ví dụ này minh họa cách JavaScript có thể được sử dụng để thực hiện các phép tính phức tạp, xử lý nhiều trường hợp khác nhau, và áp dụng kiến thức toán học vào lập trình.

Giải phương trình bậc hai

Nhập các hệ số của phương trình ax² + bx + c = 0


Giải thích

Ứng dụng: Giải phương trình bậc hai

Hàm solveQuadraticEquation() thực hiện các bước sau:

  1. Lấy giá trị của các hệ số a, b, c từ input người dùng.
  2. Kiểm tra tính hợp lệ của input.
  3. Xử lý các trường hợp đặc biệt (a = 0, phương trình bậc nhất).
  4. Tính delta (discriminant) của phương trình: Δ = b² - 4ac.
  5. Dựa vào giá trị của delta, xác định loại nghiệm và tính toán.
  6. Hiển thị kết quả cho người dùng.

function solveQuadraticEquation() {
    const a = parseFloat(document.getElementById("coef-a").value);
    const b = parseFloat(document.getElementById("coef-b").value);
    const c = parseFloat(document.getElementById("coef-c").value);
    const resultElement = document.getElementById("equation-result");

    if (isNaN(a) || isNaN(b) || isNaN(c)) {
        resultElement.innerHTML = "Vui lòng nhập đầy đủ các hệ số a, b, c.";
        return;
    }

    if (a === 0) {
        if (b === 0) {
            if (c === 0) {
                resultElement.innerHTML = "Phương trình vô số nghiệm.";
            } else {
                resultElement.innerHTML = "Phương trình vô nghiệm.";
            }
        } else {
            // Giải phương trình bậc 1: bx + c = 0
            const x = -c / b;
            resultElement.innerHTML = `Đây là phương trình bậc nhất. Nghiệm là: x = ${x.toFixed(2)}`;
        }
        return;
    }

    const discriminant = b*b - 4*a*c;

    if (discriminant > 0) {
        const x1 = (-b + Math.sqrt(discriminant)) / (2*a);
        const x2 = (-b - Math.sqrt(discriminant)) / (2*a);
        resultElement.innerHTML = `Phương trình có hai nghiệm phân biệt:
            x₁ = ${x1.toFixed(2)}
            x₂ = ${x2.toFixed(2)}`;
    } else if (discriminant === 0) {
        const x = -b / (2*a);
        resultElement.innerHTML = `Phương trình có nghiệm kép:
            x = ${x.toFixed(2)}`;
    } else {
        const realPart = -b / (2*a);
        const imaginaryPart = Math.sqrt(-discriminant) / (2*a);
        resultElement.innerHTML = `Phương trình có hai nghiệm phức:
            x₁ = ${realPart.toFixed(2)} + ${imaginaryPart.toFixed(2)}i
            x₂ = ${realPart.toFixed(2)} - ${imaginaryPart.toFixed(2)}i`;
    }
}

4. Đồng hồ thời gian thực

  1. var date = new Date();: Tạo một đối tượng Date mới, đại diện cho thời gian hiện tại.
  2. var h = date.getHours();: Lấy giờ hiện tại (0-23).
  3. var m = date.getMinutes();: Lấy phút hiện tại (0-59).
  4. var s = date.getSeconds();: Lấy giây hiện tại (0-59).
  5. var session = "AM";: Mặc định là buổi sáng (AM).
  6. Điều chỉnh giờ:
    • Nếu giờ là 0, đặt lại thành 12 (12 AM).
    • Nếu giờ lớn hơn 12, trừ đi 12 và đổi session thành "PM".
  7. Thêm số 0 đứng trước nếu giờ, phút hoặc giây nhỏ hơn 10.
  8. Tạo chuỗi thời gian dạng "HH:MM:SS AM/PM".
  9. Cập nhật nội dung của phần tử có id "clock" bằng chuỗi thời gian.
  10. **Analog Clock Hands**:
    • hourDeg: Tính toán góc quay của kim giờ, bao gồm cả ảnh hưởng của phút (30 độ mỗi giờ và 0.5 độ mỗi phút).
    • minuteDeg: Tính toán góc quay của kim phút (6 độ mỗi phút).
    • secondDeg: Tính toán góc quay của kim giây (6 độ mỗi giây).
    • Áp dụng các góc quay cho các kim đồng hồ bằng CSS transform: rotate().
  11. setTimeout(updateClock, 1000);: Đặt hẹn giờ để gọi lại hàm updateClock() sau 1 giây (1000 ms).

Hàm updateClock() được gọi lần đầu tiên để khởi động đồng hồ. Sau đó, nó tự gọi lại mỗi giây thông qua setTimeout(), tạo ra một vòng lặp liên tục cập nhật thời gian và vị trí của các kim đồng hồ.

Ứng dụng: Bộ hẹn giờ và đối tượng Date

Giải thích

Ví dụ này hiển thị một đồng hồ thời gian thực, được cập nhật liên tục bằng JavaScript. Nó bao gồm cả đồng hồ kỹ thuật số và đồng hồ analog với kim đồng hồ di chuyển.


function updateClock() {
    var date = new Date();
    var h = date.getHours(); // 0 - 23
    var m = date.getMinutes(); // 0 - 59
    var s = date.getSeconds(); // 0 - 59
    var session = "AM";
    
    if(h == 0){
        h = 12;
    }
    
    if(h > 12){
        h = h - 12;
        session = "PM";
    }
    
    h = (h < 10) ? "0" + h : h;
    m = (m < 10) ? "0" + m : m;
    s = (s < 10) ? "0" + s : s;
    
    var time = h + ":" + m + ":" + s + " " + session;
    document.getElementById("clock").innerText = time;
    document.getElementById("clock").textContent = time;
    
    // Lấy các phần tử kim đồng hồ
    const hourHand = document.getElementById("hour-hand");
    const minuteHand = document.getElementById("minute-hand");
    const secondHand = document.getElementById("second-hand");

    // Tính toán góc quay của các kim đồng hồ
    const hourDeg = h * 30 + m * 0.5;
    const minuteDeg = m * 6;
    const secondDeg = s * 6;

    // Cập nhật vị trí của các kim đồng hồ
    hourHand.style.transform = `rotate(${hourDeg}deg)`;
    minuteHand.style.transform = `rotate(${minuteDeg}deg)`;
    secondHand.style.transform = `rotate(${secondDeg}deg)`;

    setTimeout(updateClock, 1000);
}

updateClock();

5. Các hình tròn di chuyển theo vị trí con trỏ

Di chuyển chuột vào vùng dưới đây để xem hiệu ứng các hình tròn di chuyển theo vị trí con trỏ.


const updateSquarePositions = () => {
    squares.forEach((square, index) => {
        const { x, y } = cornerPositions[squarePositions[index]];
        square.style.left = `${x}px`;
        square.style.top = `${y}px`;
    });
};

const moveSquaresAwayFromCursor = (cursorX, cursorY) => {
    const availableCorners = [0, 1, 2, 3].filter(corner => !squarePositions.includes(corner));
    if (availableCorners.length === 0) return;

    const approachingSquareIndex = squarePositions.findIndex((pos) => {
        const { x, y } = cornerPositions[pos];
        const distance = Math.hypot(cursorX - (x + squareSize / 2), cursorY - (y + squareSize / 2));
        return distance < 120;
    });

    if (approachingSquareIndex !== -1) {
        const [emptyCorner] = availableCorners;
        const swapIndex = squarePositions.find((_, idx) => idx !== approachingSquareIndex);
        squarePositions[squarePositions.indexOf(swapIndex)] = emptyCorner;
        squarePositions[approachingSquareIndex] = swapIndex;
        updateSquarePositions();
    }
};

Cuối cùng, chúng ta thêm các event listener:

  • mousemove:
    • Tính toán vị trí của con trỏ trong container bằng cách trừ vị trí của container khỏi vị trí của chuột.
    • Cập nhật vị trí của phần tử cursor để theo dõi con trỏ (trừ 10px để căn giữa).
    • Gọi moveSquaresAwayFromCursor với vị trí mới của con trỏ.
  • mouseleave: Ẩn phần tử cursor khi chuột rời khỏi container.
  • mouseenter: Hiện phần tử cursor khi chuột vào container.

Hàm updateSquarePositions() được gọi ở cuối để khởi tạo vị trí ban đầu của các hình vuông khi trang web được tải.

Giải thích

Ứng dụng: Xử lí sự kiện và định vị đng cho phần tử

Ví dụ này hiển thị các hình tròn di chuyển khi con trỏ đến gần. Chúng ta sử dụng JavaScript để tính toán khoảng cách giữa con trỏ và các hình tròn, sau đó di chuyển các hình tròn đến vị trí khác nếu chúng quá gần. Dưới đây là giải thích chi tiết về từng phần của mã:


const container = document.getElementById('container');
const squares = document.querySelectorAll('.square');
const cursor = document.getElementById('cursor');

const margin = 25, containerSize = 500, squareSize = 150;
const cornerPositions = [
    { x: margin, y: margin },
    { x: containerSize - squareSize - margin, y: margin },
    { x: margin, y: containerSize - squareSize - margin },
    { x: containerSize - squareSize - margin, y: containerSize - squareSize - margin }
];

let squarePositions = [0, 1, 2];

Đoạn mã trên khởi tạo các biến và hằng số cần thiết:

  • container, squares, và cursor là các tham chiếu đến các phần tử DOM. container là phần tử chứa, squares là một NodeList chứa tất cả các hình vuông, và cursor là phần tử đại diện cho con trỏ chuột.
  • margin (25px) là khoảng cách từ cạnh container đến hình vuông, containerSize (500px) là kích thước của container, và squareSize (150px) là kích thước của mỗi hình vuông.
  • cornerPositions là một mảng chứa tọa độ của bốn góc trong container. Mỗi góc được biểu diễn bằng một đối tượng có thuộc tính x và y.
  • squarePositions là một mảng theo dõi vị trí hiện tại của ba hình vuông, sử dụng chỉ số của các góc (0, 1, 2 tương ứng với ba góc đầu tiên).

Hàm updateSquarePositions cập nhật vị trí của các hình vuông trên màn hình:

  • Nó lặp qua mỗi hình vuông trong squares.
  • Với mỗi hình vuông, nó lấy tọa độ x và y từ cornerPositions dựa trên chỉ số trong squarePositions.
  • Sau đó, nó cập nhật thuộc tính style.left và style.top của hình vuông để di chuyển nó đến vị trí mới.

Hàm moveSquaresAwayFromCursor thực hiện các bước sau:

  1. Xác định các góc còn trống bằng cách lọc ra các góc không có trong squarePositions.
  2. Nếu không có góc trống, hàm kết thúc.
  3. Tìm hình vuông gần con trỏ nhất bằng cách:
    • Lặp qua các vị trí trong squarePositions.
    • Tính khoảng cách từ tâm của mỗi hình vuông đến con trỏ sử dụng công thức Pythagoras (Math.hypot).
    • Kiểm tra xem khoảng cách có nhỏ hơn 120px không.
  4. Nếu tìm thấy hình vuông gần (khoảng cách < 120px):
    • Chọn một góc trống ngẫu nhiên.
    • Tìm mt hình vuông khác để hoán đổi vị trí.
    • Cập nhật squarePositions để phản ánh sự thay đổi vị trí.
    • Gọi updateSquarePositions() để cập nhật vị trí trên màn hình.

container.addEventListener('mousemove', (e) => {
    const { left, top } = container.getBoundingClientRect();
    const x = e.clientX - left, y = e.clientY - top;
    cursor.style.left = `${x - 10}px`;
    cursor.style.top = `${y - 10}px`;
    moveSquaresAwayFromCursor(x, y);
});

container.addEventListener('mouseleave', () => cursor.style.display = 'none');
container.addEventListener('mouseenter', () => cursor.style.display = 'block');

updateSquarePositions();
